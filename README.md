## Installation

Copy emoji.zsh-theme into your `~/.oh-my-zsh/themes/` directory

Then change current theme to emoji `ZSH_THEME=emoji` in your `~/.zshrc`.

Activate a new theme with `$ source ~/.zshrc`.
